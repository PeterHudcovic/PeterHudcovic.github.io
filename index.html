Technical Sandbox

This is my hands-on technical sandbox where I build, deploy, secure, and troubleshoot real services. What started as a Linux and web server environment has expanded into a broader infrastructure lab that now includes Kubernetes on DigitalOcean, Helm-based deployments, NGINX Ingress, Cloudflare-managed DNS and TLS, a Python FastAPI service, and a Java Spring Boot API connected to PostgreSQL through JDBC. I use this setup to practice real deployment flows, infrastructure hardening, API publishing, and application troubleshooting.

This service serves as a practical lab for deployment testing, DNS and TLS validation, ingress troubleshooting, and secure API publishing in a live hosted environment.

Java JDBC + PostgreSQL Lab

I also built and deployed a Spring Boot application connected to PostgreSQL through JDBC, giving me a working lab for database connectivity, backend integration, SQL execution, and troubleshooting across both the application and data layers. This environment is especially useful for testing connection behavior, configuration issues, and real deployment workflows for Java-based services.
